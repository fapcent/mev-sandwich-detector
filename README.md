# 🕵️‍♂️ MEV Sandwich Attack Detector

A DeFi monitoring bot connected to the Ethereum Mainnet to detect predatory trading behaviors (Maximal Extractable Value).

## 🚀 Key Features
* **Mempool Monitoring:** Real-time scanning of pending transactions via Alchemy RPC.
* **Whale Detection:** Algorithmic identification of large swaps on Uniswap V2/V3.
* **Impact Analysis:** Calculation of price slippage and potential front-running profitability.

## 🛠 Stack
* **Language:** Python
* **Blockchain:** Web3.py, Ethereum Node (Alchemy)
* **Infrastructure:** Dockerized Worker
